# The 2AM Café

A cinematic late-night café website built with pure HTML and inline CSS.

## Pages

- **Home** — The main landing page introducing the mood and identity of the café.
- **Menu** — Coffee descriptions written like character dialogue instead of products.
- **Regulars** — Profiles of strange recurring visitors who may or may not exist.
- **Tonight's Jazz** — Late-night setlists and ambient performance notes.
- **Find Us** — Directions written like a memory instead of a map.

## Built With

- HTML5
- Inline CSS
- Google Fonts

## Design Direction

Inspired by noir cinema, midnight jazz cafés, analog photography, quiet urban loneliness, vintage hotel typography, and late-night city atmosphere.
